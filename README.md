# learning-python-flashcards
A simple interactive terminal quiz app created to master fundamental Python concepts like dictionaries, loops, conditional logic, and the random module.

# Terminal Flashcard Quiz
An interactive Command Line Interface (CLI) flashcard application built using Python.

## Features
- Shuffles questions randomly every time you play.
- Case-insensitive answer checking (e.g., "Def" and "def" are both correct).
- Cleans up user input by stripping extra spaces.
- Displays a final score and success percentage.

## How to Run
1. Clone this repository or download the `flashcard_quiz.py` file.
2. Open your terminal.
3. Run the command: `python flashcard_quiz.py`
